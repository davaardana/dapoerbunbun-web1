## Cara Menambahkan:

1. Copy gambar logo ke: `public/logo-dapoer-bun-bun.jpg`
2. Copy gambar produk ke: `public/products/[nama-file].jpg`
3. Pastikan nama file PERSIS seperti di atas (huruf kecil, pakai tanda minus)
4. Format gambar harus `.jpg` atau `.jpeg`

Setelah gambar ditambahkan, website akan menampilkannya secara otomatis!
